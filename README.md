# vue-number-format

<a href="https://vue-number-format.netlify.app" rel="nofollow"><img src="docs/vue-number-format.gif" alt="" style="width: 100%;"></a>

Vue Number Format is used to format a number using fixed-point notation. It can be used to format a number with a specific number of digits to the right of the decimal.

## Installation

```bash
npm install @coders-tm/vue-number-format@2.20.0
or
yarn add @coders-tm/vue-number-format@2.20.0
```

## Features

- Lightweight
- Dependency free
- Component or Directive flavour
- Format as you type
- Built-in validation
- Supports both Vue 3 and Vue 2.
